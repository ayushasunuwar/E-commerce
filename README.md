# Full-Stack E-commerce Platform (MERN)

A modern, responsive e-commerce solution built with the MERN stack, featuring secure user authentication, product management, and integrated Stripe payments.

## ‚ú® Features
* **User Authentication:** Secure Sign-up/Login using JWT (JSON Web Tokens).
* **Product Management:** Browse products with category filtering and detailed views.
* **Shopping Cart:** Persistent cart functionality (Add/Remove/Update quantities).
* **Payment Integration:** Real-time credit card processing via **Stripe API**.
* **Order Tracking:** Dashboard for users to view their purchase history and order status.
* **Responsive UI:** Mobile-first design built with **Tailwind CSS**.

* **Payment Gateway:** Stripe API
